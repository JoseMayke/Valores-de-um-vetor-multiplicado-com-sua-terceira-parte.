# Valores de Um Vetor Multiplicado Por Sua Terceira Parte.

Uma questão comum de Estruturas de Dados I do terceiro período do curso de sistema de informação.

É um código usando duas funções e utilizando alocação dinâmica.

O algoritmo exibe todos os valores do vetor e em seguida multiplica cada posição com sua terceira parte.

Também, após a multiplicação, o algoritmo retorna cada posição do vetor multiplicado.
